# Project: SOW Compliance Agent
## Deliverables
1. An AWS Bedrock agent that can read local files.
2. A tool that identifies missing security headers in Python code.
3. A reporting tool that summarizes gaps between code and this document.

## Technical Constraints
- Must use Python 3.10+
- Must use Amazon Nova or Anthropic Claude models.
- All code must be stored in the `/src` directory.
